What Are Surds?
A surd refers to a square root that cannot be simplified into a whole or rational number.
For example ( √10 = 3.16227766017... ), this square root as a propper number will carry on forever so simplification is impossible.
An example of a square root that is not a surd is ( √9 = 3 ), this is not a surd as it can be simplified to 3. (3 x 3 = 9, √9 = 3 )
For simplifying surds you will need to simplify non-surd roots like √9 for example, so knowing your square numbers, ( 1, 4, 9, 16, 25... ), will be very importent.
Rules (multiplication and division).
For the multiplication of surds you need to know that ( √3 x √5 = √(3x5) = √15 ).
And if the surd has a multiplier you times them seperatly ( 2√3 x 4√5 = (2x4)√(3x5) = 8√15 )
You can do the same thing in reverse for division, you can split a surd into any of its factor pairs. ( √40 = √1 x √40 = √2 x √20 = √4 x √10 = √5 x √8 ). You can also do division pairs. ( √40 = √80 ÷ √2 )
*These Rules only work when multiplying and deviding surds. ( √30 + √10 ≠ √40 ❌)
The benifit of simplifying a surd into its factor pairs, ( √40 = √4 x √10 ), is that when doing so one of square roots may contain a square number, ( *in the case of √40 = √4 x √10, it will be √4 ), and this square root can be simplified to a rational intiger. ( √40 = √4 x √10 = 2 x √10, [ √4 = 2 ] ), so from splitting the surd into its factor pairs and finding one containing a square number, ( √4 ), we are able to simplify the surd. ( √40 = √4 x √10 = 2 x √10 = 2√10, [ √40 = 2√10 ] ).
If the surd is multiplied by a number, (), then when doing multiplication times the outside numbers individually. ( 2√10 x 5√12 = 10√120 [ a surd withought a number before it just has a hidden 1 ] )
One more rule is that if you multiply a surd by itself, it just returns the inner number, ( √7 x √7 = 7 ), this is because, ( √7 x √7 = √49 = 7 ).
Rules (addition and subtraction).
To add two surds together they must have the same inside number.
( √3 + √4 ≠ √7 ) ❌
( √3 + √3 = 2√3 ) ✔
( 2√3 + 5√3 = 7√3 ) ✔
The rule also applies to subtraction.
( 6√7 - 2√7 = 4√7 ) ✔
( √18 - 2√2 = √(9 x 2) - 2√2 = 3√2 - 2√2 = √2) [*Dont forget that sometimes you can simplify terms like √18 to usable numbers like √18 = 3√2]
Expanding brackets.
If you are already familiar with expanding brackets then you only really need to know how to multiply surds to understand expanding brackets with them.
lets take this example, ( 2(√2 + 3) ), just by doing your normal expanding brackets method, and aplying the rules from earlier for the surd part you can easily solve it.
2(√2 + 3) = ( 2 x √2 ) + ( 2 x 3 ) = 2√2 + 6
You can also have surds on the outside like this:
√3( √2 - 1 ) = ( √3 x √2 ) + ( √3 x 1 ) = √6 + √3
Following on from expanding brackets you can do two brackets like this:
( √7 + 2 )( √6 + √9 ) = ( √7 x √6 ) + ( √7 x √9 ) + ( 2 x √6 ) + ( 2 x √9 ) = √42 + √63 + √12 + √18 [ *This can be simplified further, try to find out how ]
