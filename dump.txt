            <div class="title">Binomial Expansion - Overview</div>
            <br>
            <div class="date">22/09/23</div>
            <br><br><br>
            <div class="header">Binomial Expansion</div>
            <div class="text">Binomial Expansion is a mathematical technique used to expand expressions of the form (a + b)<sup>n</sup>, where 'a' and 'b' are constants and 'n' is a positive integer. The result is an expression expanded into a series of terms.</div>
            <div class="text">i.e. (a+b)<sup>3</sup></div>
            <div class="text">= 1a<sup>3</sup> + 3a<sup>2</sup>b + 3ab<sup>2</sup> + 1b<sup>3</sup></div>
            <div class="text">Basically, it's a quicker method to find values like (a+b)<sup>10</sup> without multiplying it by itself 10 times.</div>

            <div class="header">How does it work?</div>
            <div class="text">Binomial Expansion uses Pascal's triangle, which you can find out more info about <a href="../pascals-triangle/index.html" class="link">here</a>, and uses the numbers found in that row as the coefficients for each expanded term.</div>
            <div class="text">Let's see an example:</div>
            <div class="text">(a+b)<sup>3</sup></div>
            <div class="text">The third row, (counting the first row as row 0), contains the numbers, (1 3 3 1)</div>
            <div class="text">This means when we expand it there will be four terms, with coefficients of 1, 3, 3, and 1</div>
            <div class="text">So next we need to add what's inside the brackets, now in this example, it's just 'a' and 'b' so it's nice and easy.</div>
            <div class="text">The first term, (regardless of the power number), will be (first term)<sup>(power num)</sup>, in this case: a<sup>3</sup></div>
            <div class="text">So knowing its coefficient is 1 because of Pascal's triangle you know the first term is 1a<sup>3</sup></div>
            <div class="text">Now going forward, the power of 'a' will decrease and 'b' will increase, look at the pattern:</div>
            <div class="text">a<sup>3</sup>b<sup>0</sup> + a<sup>2</sup>b<sup>1</sup> + a<sup>1</sup>b<sup>2</sup> + a<sup>0</sup>b<sup>3</sup></div>
            <div class="text">You can see that 'a' goes from 3 to 0, and 'b' goes from 0 to 3. If you add in the coefficients we found earlier, then you get this:</div>
            <div class="text">1a<sup>3</sup>b<sup>0</sup> + 3a<sup>2</sup>b<sup>1</sup> + 3a<sup>1</sup>b<sup>2</sup> + 1a<sup>0</sup>b<sup>3</sup></div>
            <div class="text">Then you simplify:</div>
            <div class="text">a<sup>3</sup> + 3a<sup>2</sup>b + 3ab<sup>2</sup> + b<sup>3</sup></div>

            <div class="header">More examples</div>
            <br><br>
            <div class="header">(a+b)<sup>4</sup></div>
            <div class="text">Write out the terms with ascending and descending powers:</div>
            <div class="text">a<sup>4</sup>b<sup>0</sup> + a<sup>3</sup>b<sup>1</sup> + a<sup>2</sup>b<sup>2</sup> + a<sup>1</sup>b<sup>3</sup> + a<sup>0</sup>b<sup>4</sup></div>
            <div class="text">Add coefficients from Pascal's triangle: (1 4 6 4 1)</div>
            <div class="text">1a<sup>4</sup>b<sup>0</sup> + 4a<sup>3</sup>b<sup>1</sup> + 6a<sup>2</sup>b<sup>2</sup> + 4a<sup>1</sup>b<sup>3</sup> + 1a<sup>0</sup>b<sup>4</sup></div>
            <div class="text">Simplify:</div>
            <div class="text">a<sup>4</sup> + 4a<sup>3</sup>b + 6a<sup>2</sup>b<sup>2</sup> + 4ab<sup>3</sup> + b<sup>4</sup></div>

            <div class="header">(y+d)<sup>7</sup></div>
            <div class="text">Write out the terms with ascending and descending powers:</div>
            <div class="text">y<sup>7</sup>d<sup>0</sup> + y<sup>6</sup>d<sup>1</sup> + y<sup>5</sup>d<sup>2</sup> + y<sup>4</sup>d<sup>3</sup> + y<sup>3</sup>d<sup>4</sup> + y<sup>2</sup>d<sup>5</sup> + y<sup>1</sup>d<sup>6</sup> + y<sup>0</sup>d<sup>7</sup></div>
            <div class="text">Add coefficients from Pascal's triangle: (1 7 21 35 35 21 7 1)</div>
            <div class="text">1y<sup>7</sup>d<sup>0</sup> + 7y<sup>6</sup>d<sup>1</sup> + 21y<sup>5</sup>d<sup>2</sup> + 35y<sup>4</sup>d<sup>3</sup> + 35y<sup>3</sup>d<sup>4</sup> + 21y<sup>2</sup>d<sup>5</sup> + 7y<sup>1</sup>d<sup>6</sup> + 1y<sup>0</sup>d<sup>7</sup></div>
            <div class="text">Simplify:</div>
            <div class="text">y<sup>7</sup> + 7y<sup>6</sup>d + 21y<sup>5</sup>d<sup>2</sup> + 35y<sup>4</sup>d<sup>3</sup> + 35y<sup>3</sup>d<sup>4</sup> + 21y<sup>2</sup>d<sup>5</sup> + 7yd<sup>6</sup> + d<sup>7</sup></div>

            <div class="header">Proper Usage</div>
            <div class="text">So expanding terms like (a+b)<sup>c</sup> or (y+d)<sup>n</sup> is relatively easy. All you need to do is write the term out in ascending and descending order with Pascal's triangle for coefficients, but real problems won't just have a+b or y+d</div>
            <div class="text">An example is this (2y + 1)<sup>3</sup>, now you do this in the exact same way as you would do (a+b)^3, just with a=2y and b=1.</div>
            <div class="text">Now when you have a question that's not just a+b etc., you have 2 choices for what to do. Either you can do (a+b)<sup>power in question</sup>, then after simplifying you just substitute 'a' for the first term and 'b' for the second.</div>
            <div class="text">i.e. (2y+1)<sup>3</sup></div>
            <div class="text">a+b: (a+b)<sup>3</sup> = a<sup>3</sup> + 3a<sup>2</sup>b + 3ab<sup>2</sup> + b<sup>3</sup></div>
            <div class="text">Substitute: (2y)<sup>3</sup> + 3(2y)<sup>2</sup>(1) + 3(2y)(1)<sup>2</sup> + (1)<sup>3</sup></div>
            <div class="text">Simplify: 8y<sup>3</sup> + 12y<sup>2</sup> + 6y + 1</div>
            <div class="text">The second method is just straight up using the terms when writing it out, basically skipping the (a+b) step, but this is more complicated, so I would recommend just doing (a+b) and substituting.</div>

            <div class="header">Even more examples:</div>
            <br><br>
            <div class="header">(3f - 1)<sup>4</sup></div>
            <div class="text">(a+b) terms: a<sup>4</sup> + a<sup>3</sup>b + a<sup>2</sup>b<sup>2</sup> + ab<sup>3</sup> + b<sup>4</sup></div>
            <div class="text">(a+b) coefficients: 1a<sup>4</sup> + 4a<sup>3</sup>b + 6a<sup>2</sup>b<sup>2</sup> + 4ab<sup>3</sup> + 1b<sup>4</sup></div>
            <div class="text">(a+b) simplify: a<sup>4</sup> + 4a<sup>3</sup>b + 6a<sup>2</sup>b<sup>2</sup> + 4ab<sup>3</sup> + b<sup>4</sup></div>
            <div class="text">Substitute terms: (3f)<sup>4</sup> + 4(3f)<sup>3</sup>(-1) + 6(3f)<sup>2</sup>(-1)<sup>2</sup> + 4(3f)(-1)<sup>3</sup> + (-1)<sup>4</sup></div>
            <div class="text">Simplify: 81f<sup>4</sup> - 108f<sup>3</sup> + 54f<sup>2</sup> - 12f + 1</div>

            <div class="header">(a<sup>2</sup> + 1)<sup>3</sup></div>
            <div class="text">(a+b) terms: a<sup>3</sup> + a<sup>2</sup>b + ab<sup>2</sup> + b<sup>3</sup></div>
            <div class="text">(a+b) coefficients: 1a<sup>3</sup> + 3a<sup>2</sup>b + 3ab<sup>2</sup> + 1b<sup>3</sup></div>
            <div class="text">(a+b) simplify: a<sup>3</sup> + 3a<sup>2</sup>b + 3ab<sup>2</sup> + b<sup>3</sup></div>
            <div class="text">Substitute terms: (a<sup>2</sup>)<sup>3</sup> + 3(a<sup>2</sup>)<sup>2</sup>(1) + 3(a<sup>2</sup>)(1)<sup>2</sup> + (1)<sup>3</sup></div>
            <div class="text">Simplify: a<sup>6</sup> + 3a<sup>4</sup> + 3a<sup>2</sup> + 1</div>
